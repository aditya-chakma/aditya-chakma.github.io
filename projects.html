Mentionable Course Projects

  • C++: Dx-ball [Github]

    The 90's classic DX-Ball game re-created using C++, OpenGL, and iGraphics library. This game has 5 different levels, a lives system, and a scoring system. Can you beat my high score?

    • Tech Stack:
      C++, iGraphics, OpenGL/Glut Library
    • Features:
      5 levels, Scoring system, Lives system, Music
  • Java: Blood Bank [Github]

    Used JavaFx to build a blood donation desktop application. Users and donors can register in the system. Upon a user's request for blood, donors in the corresponding location will get notified.

    • Tech Stack:
      Java, JavaFx, CSS
    • Features:
      Donor registration, User registration, Location-based donor notification, Requester notification on donor confirmation.
  • Software Engineering: Travel Mate [Github]

    Travel Mate is a web application designed for managing tour agencies. It caters to two primary user groups: business owners and tourists. Business owners can register their businesses and list services such as hotel bookings and vehicle rentals. Tourists can sign up to explore and book these services.

    • Tech Stack:
      Laravel, MySql
    • Features:
      User registration, Client Registration, Product search, Product payment, Room/Vehicle rent, release.
  • Neural Networks: LCU-Net [Github]

    U-Net is an encoder-decoder based CNN architecture primarily designed for EMs (Environmental micro-organisms) segmentation. LCU-Net is a version of U-Net using much lower memory and computational power. I implemented the paper LCU-Net: A novel low-cost U-Net for environmental microorganism image segmentation from scratch in Python.

    • Tech Stack:
      Python, Tensorflow, Keras
  • Miscellaneous: ARP cache poisoning, Arithmetic Logical Unit (ALU), 4 bit computer, Remote Control Car using ATMega32, etc.

Personal Projects

  • Attention (Google Chrome extension) [Github]

    A Google Chrome extension meant for keeping you from distractions! Bookmark timestamps for a YouTube video, remove YouTube shorts, remove Facebook reels, and block websites.

    • Tech Stack:
      JavaScript, HTML, CSS, ManifestV3
    • Features:
      Bookmark timestamps of YouTube videos, remove Facebook reels from newsfeed, remove YouTube shorts from homepage, block websites.
  • Online Shop [Github]

    An e-commerce online shop website. Users can open their shop and sell products online by posting them. After site admin validates and accepts the products, they will be visible to customers. Customers can place orders for the products.

    • Tech Stack:
      Java, Spring, Hibernate, JDBC, MySql, HTML, CSS, JSP
    • Features:
      • Admin
        Login, Add Product, Discontinue Product, Update Product, See all Products, Confirm Order, Reject Order, Search Product by name
      • Customer
        Register, See all Products, Add Product to Cart, Checkout Cart, Place Order, Delete Order, View previous Order history, Search product by name

  • Multilingual machine translation for Chakma language
